# 사장톡 - 자영업자 커뮤니티 앱 Blueprint

## Overview

'사장톡' is a mobile-first web application designed as a community hub for self-employed business owners. It provides a platform for communication, trading, and seeking advice. The user interface is modeled after popular apps like Karrot Market (당근마켓) and KakaoTalk to offer a familiar and intuitive experience.

## Features

*   **Mobile-First Design:** The layout is optimized for a maximum width of 480px, simulating a native mobile app experience.
*   **Home Feed:** The main screen displays a summary of hot posts and local marketplace items.
*   **Tab-Based Navigation:** A fixed bottom navigation bar allows easy switching between different sections: Home, Community, Marketplace, Consulting, and Profile.
*   **Dynamic Content Rendering:** The app uses JavaScript to render data and simulate Single Page Application (SPA) behavior by dynamically changing the content without full page reloads.
*   **Mock Data:** The initial version uses mock data to simulate API responses for posts and market items, allowing for rapid front-end development.

## Design and Style

*   **Color Palette:** Uses an orange accent color (`#ff6f0f`), reminiscent of Karrot Market, for key branding elements.
*   **Typography:** Employs the 'Pretendard' font for a clean and modern look.
*   **Layout:** A sticky header and a fixed bottom navigation bar provide persistent controls, while the main content area is scrollable.
*   **Card UI:** Information is presented in clean, well-structured cards with clear typography and spacing.
